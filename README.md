# 🚀 AI Productivity Tracker

🧠 A smart web application that helps students track daily habits, analyze productivity, and improve performance using behavioral insights.

---

## 📌 Project Description

The **AI Productivity Tracker** is designed to bridge the gap between daily lifestyle habits and personal productivity.
Instead of relying on unrealistic academic metrics like GPA, this system evaluates **real-life behaviors** such as sleep, study time, screen usage, exercise, diet, and stress.

It generates a **daily productivity score (0–10)** and continuously updates insights based on historical data, helping users build better habits and improve consistency over time.

## 🧠 How It Works

1. User enters daily lifestyle data
2. System calculates a **Productivity Score**
3. Data is stored in a database
4. Dashboard updates with:

   * 📊 Trends
   * 📈 Insights
   * 🎯 Overall performance

## 🚀 How to Run

